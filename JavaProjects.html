Programs

Partial Search Using Inverted Index and JSON Format

Created a program that takes in a directory finds all .txt files. Text files are parsed and an inverted index of words, positions, and relative paths are stored using multiple nested data structures. The inverted index is then output into JSON format. The program then takes in a .txt of query words and returns a sorted list of search results. List is sorted based on relevance (frequency and position).

Word Counter Using Binary Search Tree

Created a program that will read in a text file and create a Binary Search Tree of all the words found in that text file with a count for how many times each word appears. There is also a list of stop words such as “a, the, and, or, etc.” that will be ignored and not added to the Binary Search Tree. Includes functions that would allow the user to find a word in the Binary Search Tree, remove a word, find the parent node of an existing word, or print the tree using in order, postorder, and preorder traversal.

Photo Album Viewer

Created a GUI that allowed a user to load in a collection of photographs. Then, the user could recursively traverse through this album (forwards and backwards) to view the imported photos. The photos can also automatically traverse forwards using a timer. There is also a function that allows the user to print the title of each image into a .txt file.

Game of Blackjack

Created a project that simulates the game of blackjack. The dealer and player are each dealt two cards to start the game. The player is given the choice to hit or stay with an Ace being a 1 or an 11. The dealer will hit until he reaches a soft 17.
